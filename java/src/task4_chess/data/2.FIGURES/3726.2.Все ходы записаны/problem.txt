3.Все ходы записаны

Создать и заполнить структуру данных, в которой хранятся все возможные ходы из всех полей шахматной доски для всех фигур, кроме короля и пешек.

Это массив из 256 элементов, 
индекс которого в двоичном виде определяется следующим образом:
ffyyyxxx
ff - номер фигуры, 00 - конь, 01 - слон, 10 - ладья 11 - ферзь
yyy - № горизонтали поля, от 1 до 8
xxx - № вертикали поля, от a до h.

Каждый элемент массива хранит ссылку на список, в котором перечислены все возможные ходы выбранной фигурой из выбранного поля.

Элемент списка обладает следующей структурой:

class MoveNode {
  int coord - координата, куда фигура может пойти
  MoveNode next - ссылка на следующий ход или null
  MoveNode turn - ссылка на следующую группу ходов
}

Написать программу, которая создаст указанный массив и заполнит его данными.

Программа должна вывести количество возможных ходов 
для указанного элемента массива в указанном формате.

Пример: 0 = 00 000 000 - конь, стоящий на поле a1
Ответ: 2
Список:
[ 0: { b3, #1, #1 } , 1: { c2, null, null }]

Пример: 73 = 01 001 001 - слон, стоящий на поле b2
Ответ: 9
Список:
[
  0: { a1, #1, #1 }, 
  1: { a3, #2, #2 }, 
  2: { c3, #3, #8 }, 3: { d4, #4, #8 }, 4: { e5, #5, #8 }, 
  5: { f6, #6, #8 }, 5: { g7, #7, #8 }, 6: { h8, #8, #8 }, 
  8: { c1, null, null } ]
]
