2.Сортировка ходов

Результатом работы последующих задач будет список ходов.
Чтобы проще было проверять ответы - нужно договориться о порядке вывода ходов.

Каждый ход состоит из 4 или 5 символов.
1-2 символ - координата начальной клетки, 
3-4 символ - координата конечной клетки, 
5 символ - фигура превращения, если это ход пешкой на последнюю горизонталь.

Ходы сначала сортируются по алфавиту, а потом группируются по начальным координатам.
Каждая группа выводится на отдельной строчке через пробел, в конце строки пробела не должно быть.

Дано: список ходов. На первой строчке записано число N - количество ходов.
      далее, на N строчках записаны ходы, по одному на каждой строке.
      ходы могут следовать в любом порядке.
Надо: отсортировать список ходов по алфавиту 
      сгруппировать по начальной координате
      вывести количество ходов, а затем их список, 
      каждая группа на отдельной строке через пробел.
